# 👗 X-Fashion Website

A beautiful fashion website with lavender theme 🌸

## 📖 About

This project was created as **Task 1** for my **Elevvo internship** 💼

## ✨ Features

- 📱 Responsive design
- 🎯 Interactive sidebar
- 👚 Product gallery
- 💜 Lavender theme
- ⚡ Smooth animations

- HTML5
- CSS3
- JavaScript
- Icons & Fonts

1. Download files 📂
2. Open `index.html` 🌐
3. Enjoy! ✅

- `script.js` - Functionality
